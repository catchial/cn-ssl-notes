# cn-ssl-notes

## 项目简介

本项目是一个基于 GitHub Pages 的静态页面归档仓库，主要用于存放和发布与 SSL/TLS 相关的独立 HTML 页面。这些页面内容围绕中国互联网环境下的 SSL 证书、加密协议、证书颁发机构（CA）等主题展开，以笔记、参考、说明等形式呈现。

仓库本身不针对任何特定网站或域名，而是以通用资料页面的方式归档，供有需要的人查阅或引用。

## 目录说明

```
cn-ssl-notes/
├── index.html          # 仓库主页，展示归档页面列表
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/` 目录下每个 HTML 文件是一个独立页面，内容互不依赖。
- `assets/` 存放页面共用的样式、脚本或媒体文件，便于统一维护。
- 主页 `index.html` 提供页面导航和简要说明。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开，无需后端支持。
- 页面内容以 SSL/TLS 笔记、技术说明、参考信息为主，不包含任何商业推广或诱导点击内容。
- 归档页面可能不定期更新或补充，历史版本可通过 Git 提交记录追溯。
- 页面中不会出现具体的域名、网址或指向外部服务的链接，确保内容的通用性和中立性。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交建议、报告问题或讨论相关内容。
- 不接受直接向仓库推送未经审核的 Pull Request，如需贡献请先通过 Issues 沟通。
